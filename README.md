# Closet Virtual App - SDK 54

Proyecto base en Expo SDK 54, adaptado al diseño de tu Figma para que funcione con Expo Go en celular.

## Pantallas incluidas
- Splash
- Login
- Register
- Home
- Wardrobe / Armario
- Add Garment
- Outfits
- Create Outfit
- Profile

## Cómo correrlo
```bash
npm install
npx expo start
```

Si Expo detecta algún desajuste de paquetes, ejecuta:
```bash
npx expo install --fix
```
